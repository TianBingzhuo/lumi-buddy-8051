# 硬件说明

## 目标平台

- 普中 51 / STC89C52 兼容课程板。
- LCD12864 图形点阵液晶，ST7920 类控制器，并口模式。
- DS1302 实时时钟模块。
- DS18B20 温度接口保留（当前版本未启用）。

## 接线表

| 模块 | 单片机引脚 |
| --- | --- |
| LCD12864 `DB0-DB7` | `P0.0-P0.7` |
| LCD12864 `RS` | `P2.6` |
| LCD12864 `RW` | `P2.5` |
| LCD12864 `E` | `P2.7` |
| LCD12864 `PSB` | `P3.2` |
| DS1302 `IO` | `P3.4` |
| DS1302 `RST/CE` | `P3.5` |
| DS1302 `CLK/SCLK` | `P3.6` |
| DS18B20 `DQ` | 保留接口，原课件使用 `P3.7` |

## 温度预留说明

调试过程中发现课程板的 `P3.7/DQ` 在拔掉 DS18B20 之后仍然读到低电平，所以提交版本启用了 `TEMP_RESERVED_MODE`：

- 不访问 DS18B20 总线；
- 在温度区域显示模拟温度值；
- DS18B20 的相关代码全部保留在源码里，硬件恢复正常后关掉预留宏就能重新启用实测温度。

## 按键预留说明

课程资料里一直没有给出稳定的板载按键映射，加上 `P3.2` 已经被 LCD12864 的 `PSB` 占用了，所以当前版本没有做按键扫描。将来确认了实板按键映射之后，计划把按键用于这些功能：

- 切换头像表情状态；
- 触发挥手动画；
- 返回自动状态；
- 切换 `CAR MODE` / `ROS MODE` 等状态面板页面。
