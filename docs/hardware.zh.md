# 硬件说明

## 目标平台

- 普中 51 / STC89C52 兼容课程板。
- LCD12864 图形点阵液晶，ST7920 类控制器，并口模式。
- DS1302 实时时钟模块。
- DS18B20 温度接口保留。

## 接线表

| 模块 | 单片机引脚 |
| --- | --- |
| LCD12864 `DB0-DB7` | `P0.0-P0.7` |
| LCD12864 `RS` | `P2.6` |
| LCD12864 `RW` | `P2.5` |
| LCD12864 `E` | `P2.7` |
| LCD12864 `PSB` | `P3.2` |
| DS1302 `IO` | `P3.4` |
| DS1302 `RST/CE` | `P3.5` |
| DS1302 `CLK/SCLK` | `P3.6` |
| DS18B20 `DQ` | 保留接口，原课件使用 `P3.7` |

## 温度预留说明

实验调试中发现课程板 `P3.7/DQ` 在拔掉 DS18B20 后仍读低电平，因此提交版本启用 `TEMP_RESERVED_MODE`：

- 不访问 DS18B20 总线；
- 在温度区域显示模拟温度；
- 保留 DS18B20 相关代码，硬件恢复后可关闭预留模式重新启用实测温度。

## 按键预留说明

课程资料没有给出稳定的板载按键映射，并且 `P3.2` 已被 LCD12864 的 `PSB` 占用。当前演示版本不扫描按键。未来版本可在确认实板映射后，将按键用于：

- 切换头像状态；
- 触发挥手动画；
- 返回自动状态；
- 切换 `CAR MODE` / `ROS MODE` 等状态面板页面。
